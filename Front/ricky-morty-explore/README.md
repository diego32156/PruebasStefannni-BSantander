# Rick & Morty Explorer

Aplicación frontend desarrollada con Angular 17+, Signals y Tailwind CSS.  
Permite listar personajes de Rick and Morty, buscar en tiempo real por nombre y visualizar información básica como imagen, especie y estado.

## Tecnologías usadas

- Angular 17+
- TypeScript
- Signals
- computed
- toSignal
- Tailwind CSS
- Rick and Morty API

## Requisitos previos

Tener instalado:

- Node.js
- Angular CLI
